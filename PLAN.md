# Universal Pro + App 3.0 유저 가이드 제작 계획

> 기준: 기존 7개국어 유저 가이드 (App 2.0) → App 3.0 + Season 2 Universal Pro 버전으로 전면 개정
> 작성일: 2026-04-13

---

## 전체 변경 방향 요약

| 항목            | 기존 (App 2.0)           | 신규 (App 3.0)                              |
| --------------- | ------------------------ | ------------------------------------------- |
| 회사명          | Artisan&Ocean            | OceanWick                                   |
| 앱 버전         | DIVEROID App 2.0         | DIVEROID App 3.0                            |
| Mini 의존성     | Mini 필수 (Bluetooth ON) | Mini 없이 단독 사용 가능 (사진/영상/로그북) |
| 다이빙 컴퓨터   | Mini 내장 센서           | Mini 있을 때만 활성 (별도 명시)             |
| 스마트폰 설정표 | Mini 유무 2열 비교       | App 3.0 단독/Mini 연결 2모드로 재구성       |
| 앱 UI 이미지    | App 2.0 스크린샷         | **Figma에서 App 3.0 화면으로 교체 필요**    |
| 구성품          | Housing + Mini 세트 기준 | Housing 단독 기준 (Mini 별매 표기)          |

### 2장. DIVEROID App 사용 (Using DIVEROID App)

#### 2-1. Installing DIVEROID App (앱 설치)

- ✅ 유지: Google Play / App Store 설치 안내
- ✅ 유지: 회원가입 / 로그인
- ✅ 유지: 디바이스 선택
- ✅ 유지: 시리얼 번호 등록 절차
- 🔄 수정: DIVEROID 앱 아이콘 이미지 → **App 3.0 아이콘으로 교체**
- 🔄 수정: "Universal Housing and Mini can be used after product registration" 문구 → Mini 언급 제거, "Universal Housing을 등록 후 바로 사용 가능" 으로 수정
- 📷 이미지 필요 (Figma): **App 3.0 아이콘 / 앱 스토어 배너**

#### 2-2. Setting a Smartphone (스마트폰 설정)

- ❌ 삭제: 기존 Bluetooth ON/OFF 비교표 (Mini 기준이라 전면 재작성)
- 🆕 신규 작성: App 3.0 기준 2가지 모드 설정표

  | 설정             | Universal Pro 단독 | Universal Pro + Mini |
  | ---------------- | ------------------ | -------------------- |
  | Bluetooth        | OFF                | ON                   |
  | Airplane mode    | ON                 | ON                   |
  | Alarm & Calendar | OFF                | OFF                  |
  | Battery          | 충전 권장          | 충전 권장            |

- ✅ 유지: 스마트폰 케이스 제거 안내
- 📷 이미지 필요 (Figma): **App 3.0 설정 화면 스크린샷**

### 3장. 제품 사용 (Using the Product)

#### 3-1. Preparing for Use (사용 준비)

- ❌ 삭제: "In case of Universal+Mini, run the app and turn on the switch of Mini" 문구 및 Mini 관련 모든 주석

- 📷 이미지 필요 (Figma): **App 3.0 홈화면, 카메라 화면, 다이빙 시작 화면**

#### 3-2. Operating the Buttons (버튼 조작)

- ✅ 유지: Top / Center / Bottom 버튼 기본 기능 (하드웨어 동일 가정)
- 🔄 확인 필요: **App 3.0에서 버튼 기능 변경 사항 있는지 루피 확인**

- 📷 이미지: 하드웨어 버튼 다이어그램 → 기존 재사용 가능 (변경 없으면)
